# Waketooth
 automated installer for Bluetooth fix after sleep (Bazzite / SteamOS Game Mode)

## Usage

Save the script as install-bluetooth-fix.sh.

### Make it executable:
```
chmod +x install-bluetooth-fix.sh
```
### Run it with root privileges:
```
sudo ./install-bluetooth-fix.sh
```
After that, every time your system wakes from sleep (including SteamOS Game Mode), the Bluetooth fix will run automatically.
